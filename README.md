# Praticando CSS-SASS

## Live SASS Compiler
- **Extensões**
	- Para utilizar, quando tiver o arquivo `.scss/.SASS` principal criado, você pode utilizar o atalho **‘ctrl’ + ‘shift’ + ‘p’** e clicar na opção *‘Live SASS: Watch SASS’*, ou ir na barra inferior e clicar no botão *‘Watch SASS’*.
- `npm`
	- `npm install -g sass`
	- `SASS –version`
